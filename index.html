SPX90 reverse early reflections
for the way people actually use it.

SPX90 Reverse ER2 is a JUCE-based reverb plugin that recreates the Yamaha SPX90 Reverse Early Reflections 2 algorithm using 451 hardware impulse responses, tuned for washed-out, guitar-forward mixes.

Download macOS AU / VST3 Read more

Currently macOS only. Windows VST3 build is planned.

Plugin

Reverse ER2, captured setting by setting.

Every combination of Room Size (0.1–20.0) and Liveness (0–10) from the SPX90 Reverse Early Reflections 2 program is sampled as a 44.1 kHz stereo impulse response, using filenames like SPX90_ER2_REV_RS01_LV8_44p1k.wav.

The plugin uses a simple chain:

  • Input gain
  • Mono → stereo (if needed)
  • Wet-only pre-delay
  • Convolution with the selected IR
  • +1.5 dB high-shelf at 8 kHz (wet only)
  • Custom dry / wet mix curve
  • Output gain

Controls are kept to six knobs: Input Gain, Room Size, Liveness, Pre-delay, Mix, Output Gain. The mix curve keeps the first 0–25% mostly dry, then blooms quickly into the “wall of reverse” zone from 30–100%.

Info

Requirements & format.

Host · Any DAW that supports Audio Units (AU) or VST3.

macOS · 10.13 High Sierra or later, Intel or Apple Silicon.

Sample rate · Impulses captured at 44.1 kHz; the host handles resampling.

Installation · AU to /Library/Audio/Plug-Ins/Components, VST3 to /Library/Audio/Plug-Ins/VST3. IRs are bundled inside each plugin at Contents/Resources/IRs/.
